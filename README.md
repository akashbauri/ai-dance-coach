# AI Dance Coach PRO (Final) 🕺
Real-time AI pose analysis with Multi-Auth Login.

## Setup
1. Update `app.js` with your Firebase Keys.
2. Add your Vercel URL to Authorized Domains in Firebase.
3. Deploy to Vercel.

## Core Files
- index.html (UI)
- app.js (AI & Auth)
- style.css (Design)
- vercel.json (Routing)
